요구사항 확인

2020 정보처리 기사 실기 요구사항 확인을 공부하면서 꼭 외워야 할 것을 주관식으로 만들어 보았습니다.


요구사항 개발 프로세스 4가지 절차를 적으시오!
Text 정답확인 도출-분석-명세-확인

요구사항을 보다 쉽게 이해할 수 있도록 현실 세계의 상황을 단순화하여 개념적으로 표현한 것을 무엇이라고 하는가?
Text 정답확인 개념 모델링(Conceptual Modeling)

구문(Syntax)과 의미(Semantics)를 갖는 정형화된 언어를 이용해 요구사항을 수학적 기호로 표현한 후 이를 분석하는 과정을 무엇이라고 하는가?
Text 정답확인 정형 분석(Formal Analysis)

요구사항 분석에서 자료의 흐름 및 변환 과정을 기능을 도형 중심으로 기술하는 방법으로 자료 흐름 그래프, 또는 버블 차트라고 하는 이것을 무엇이라고 하는가?
Text 정답확인 자료 흐름도(DFD:Data Flow Diagram)

상품이나 서비스가 출시되기 전에 개발 대상 시스템 또는 그 일부분을 개약적을 만든 원형을 무엇이라고 하는가?
Text 정답확인 프로토타입

시스템이나 객체들이 메시지를 주고 받으며 시간이 흐름에 따라 상호 작용하는 과정을 그림으로 표현한 것을 무엇이라고 하는가?
Text 정답확인 시퀀스 다이어그램

운영체제와 소프트웨어 애플리케이션 사이의 소프트웨어 애플리케이션으로서 운영체제가 제공하는 서비스를 추가 및 확장하여 제공하는 컴퓨터 소프트웨어를 무엇이라고 하는가?
Text 정답확인 미들웨어(Middleware)

동적인 웹 사이트, 웹 애플리케이션, 웹 서비스의 개발을 지원하기 위하여 설계된 소프트웨어로서 데이터 접근, 세션 관리, 트랜잭션 관리 등을 위한 라이브러리를 제공하고 있는 것을 무엇이라고 하는가?
Text 정답확인 웹 애플리케이션 서버(WAS: Web Application Server)

사용자, 다른 애플리케이션, 데이터베이스와 상호 작용하여 데이터를 저장하고 데이터베이스를 생성, 조회, 변경하기 위한 컴퓨터 소프트웨어 애플리케이션을 무엇이라고 하는가?
Text 정답확인 DBMS

하드웨어와 소프트웨어 리소스를 관리하고 컴퓨터 프로그램을 위한 공통 서비스를 제공하는 소프트웨어를 무엇이라고 하는가?
Text 정답확인 운영체제(OS: Operating System)

다양한 하드웨어 및 운영체제에서 자바언어로 작성된 애플리케이션을 수행하기 위한 사양의 구현체를 의미 하는 이것을 무엇이라고 하는가?
Text 정답확인 자바 가상 머신(JVM: Java Virtual Machine)

소스 코드를 공개해 누구나 특별한 제한 없이 그 코드를 보고 사용할 수 있는 오픈 소스 라이선스를 만족하는 소프트웨어를 무엇이라고 하는가?
Text 정답확인 오픈 소스(Open Source)

웹 애플리케이션 서버(WAS: Web Application Server)의 종류를 3가지 이상 쓰시오.
Text 정답확인 GlassFish, JBoss, Jetty, JEUS, Resin, WebLogic, WebSphere

요구공학(Requirements Engineering)이란 요구사항을 정의하고, 문서화하고, 관리하는 프로세스를 의미한다. 이 프로세스의 4가지 개발 프로세스를 적으시오.
Text 정답확인 도출(Elicitation), 분석(Analsysis), 명세(Specification), 확인(Validation)

소프트웨어가 해결해야 할 문제를 이해하는 첫 번째 단계로서 요구사항이 어디에 있고, 어떻게 수집할 것인가와 관련되어 있으며, 이해관계자(Stakeholder)가 식별되고, 개발 팀과 고객 사이의 관계가 만들어지며 다양한 이해관계자와 효율적인 의사소통이 중요한 요구사항 프로세스 개발 단계는?
Text 정답확인 요구사항 도출(Requirement Elicitation)

시스템의 정적 구조(Static Structure)와 다양한 추상화 및 구현 수준에서 시스템의 구성요소, 구성 요소들간의 관계를 보여주고, 사용 시나리오를 나타내기 위하여 유스케이스 다이어그램을 많이 사용하는 요구사항 분석 기법은?
Text 정답확인 UML 다이어그램

요구사항 분석의 마지막 단계에서 이루어지며, 형식적으로 정의된 시맨틱(Semantics)을 지닌 언어로 요구사항을 표현하고, 정확하고 명확하게 표현하여 오해를 최소화시킬 수 있는 분석 기법은?
Text 정답확인 정형 분석(Formal Analysis)

현실 세계 문제에 대한 모델링이 소프트웨어 요구사항 분석의 핵심이며, 모델은 문제가 발생하는 상황에 대한 이해를 증진시키고 해결책을 설명하는 요구사항 분석 기법을 쓰시오.
Text 정답확인 개념 모델링(Conceptual Modeling)

새로운 요구사항을 도출하기 위한 수단으로, 소프트웨어 요구사항에 대해 소프트웨어 엔지니어가 해석한 것을 확인하기 위한 수단으로 많이 사용하는 요구사항 확인 기법은?
Text 정답확인 프로토타이핑(Prototyping)
