제품 소프트웨어 패키징

2020 정보처리 기사 실기를 공부하면서 꼭 외워야 할 것을 주관식으로 만들어 보았습니다. 참고만 하세요^^


개발 과정에서 정리된 릴리즈 정보를 소프트웨어의 최종 사용자인 고객과 공유하기 위한 문서를 무엇이라고 하는가?
Text 정답확인 릴리즈 노트(Release Note)

소프트웨어의 개발과정에서 소프트웨어의 변경 사항을 관리하기 위해 개발된 일년의 활동을 무엇이라고 하는가?
Text 정답확인 형상관리(SCM:Software Configuration Management)

리누즈 토발즈(Linus Torvalds)가 2005년 리눅스 커널 개발에 사용할 관리 도구로 개발한 이후 주니오 하마노(Junio Hamano)에 의해 유지 보수되고 있는 소트프웨어 버전 관리 도구는 무엇인가?
Text 정답확인 git

JAVA 기반의 오픈소스 형태로, 가장 많이 사용되는 빌드 자동화 도구는 무엇인가?
Text 정답확인 Jenkins

Groovy를 기반으로 한 오픈 소스 형태의 자동화 도구로 안드로이드 앱 개발 환경에서 사용되는 빌드 자동화 도구는 무엇인가?
Text 정답확인 Gradle

소프트웨어 설계에서 기능단위로 분해하고 추상화 되어 재사용 및 공유 가능한 수준으로 만들어진 단위를 무엇이라고 하는가?
Text 정답확인 모듈

소프트웨어의 성능을 향상시키거나 시스템의 디버깅, 시험, 통합 및 수정을 용이하도록 하는 소프트웨어 설계 기법을 무엇이라고 하는가?
Text 정답확인 모듈화

소스 코드 파일을 컴퓨터에서 실행할 수 있는 제품 소프트웨어의 단위로 변환하는 과정을 말하거나 그에 대한 결과물을 무엇이라고 하는가?
Text 정답확인 소프트웨어 빌드(Software Build)

개발 과정에서 소프트웨어가 얼마나 개선되었는지를 정리한 정보를 사용자와 공유하기 위해 작성하는 문서로, 이를 통해 사용자는 소프트웨어에 포함된 서비스나 사용 환경을 확인 할 수 있는 것을 무엇이라고 하는가?
Text 정답확인 릴리즈 노트
디지컬 저작권 관리 과정에서 저작권에 대한 사용 권한, 라이선스 발급, 사용량에 따른 결제 관리 등을 수행하는 곳은?
Text 정답확인 클리어링 하우스(Clearing House)

디지컬 저작권 관리 과정에서 콘텐츠의 메타데이터와 함계 배포 가능한 형태로 묶어 암호화하는 프로그램은?
Text 정답확인 패키저(Packager)

소프트웨어 작업 단계별 버전 등록 기법을 순서대로 작성하시오.(인출,예치,동기화,차이,추가)
Text 정답확인 추가(Add)-인출(Check-out)-예치(Commit)-동기화(Update)-차이(Diff)

소프트웨어 개발 과정에서 기본적으로 작성된 패키지의 변경 내용을 관리하기 위해 개발된 일련의 활동을 무엇이라고 하는가?
Text 정답확인 형상관리

기존 리눅스 커널의 버전 컨트롤을 하는 Bitkeeper를 대체 하기 위해서 나온 새로운 버전 컨트롤로 현재의 리눅스는 이것을 통해 버전 컨트롤이 되고 있다. 이것은 무엇인가?
Text 정답확인 Git

개발 완료한 파일 수정 이후 로컬 저장소에 커밋한 이후 다시 원격 저장소에 반영하는 방식의 버전 방식을 무엇이라고 하는가?
Text 정답확인 분산 저장소 방식

버전관리는 방식에 따라 3가지로 나눌 수 있다. 3가지를 작성하시오!
Text 정답확인 공유 폴더 방식, 클라이언트/서버 방식, 분산 저장소 방식
