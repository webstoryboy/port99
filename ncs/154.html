Text 정답확인 커널(Kernel)

사용자의 명령어를 인식하여 프로그램을 호출하고 명력을 수행하는 명력어 해석기를 무엇이라고 하는가?
Text 정답확인 쉘(Shell)

Text 정답확인 NUR(Not Used Recently)

이미 할당된 CPU를 다른 프로세스가 강제로 빼앗아 사용할 수 없는 스케줄링 기법을 무엇이라고 하는가?
Text 정답확인 비선점(Non-preemptive) 스케줄링

시스템에서 측정 프로세스의 우선순위가 낮아 무한정 기다리게 되는 경우, 한 번 양보하거나 기다린 시간에 비례하여 일정 시간이 지나면 우선 순위를 한 단계식 높여 가까운 시간 안에 자원을 할당받도록 하는 기법을 무엇이라고 하는가?
Text 정답확인 에이징(Aging) 기법

Text 정답확인 데이터 웨어하우스(Data Warehouse)

전사적으로 구축된 데이터 웨어하우스로부터 측정 주제나 부서 중심으로 구축된 소규모 단일 주제의 데이터 웨어 하우스를 무엇이라고 하는가?
Text 정답확인 데이터 마트(Data Mart)

데이터 웨어하우스에 저장된 집합에서 사용자의 요구에 따라 유용하고 가능성 있는 정보를 발견하기 위한 기법을 무엇이라고 하는가?
Text 정답확인 데이터 마이닝(Data Mining)

다차원으로 이루어진 데이터로부터 통계적인 요약 정보를 분석하여 의사 결정에 활용하는 방식을 무엇이라고 하는가?
Text 정답확인 OLAP(Online Analytical Processing)

온라인 업무 처리 형태의 하나로 네트워크상의 여러 이용자가 실시간으로 데이터베이스의 데이터를 갱신하거나 검색하는 등의 단위 작업을 처리하는 방식을 무엇이라고 하는가?
Text 정답확인 OLTP(Online Transaction Processing)

한 릴레이션 내에 있는 속성들의 집합으로 구성된 키로, 릴레이션을 구성하는 모든 튜풀에 대해 유일성은 만족하지만 최소성은 만족하지 못하는 키의 개념은 무엇인가?
Text 정답확인 슈퍼키(Super Key)

Text 정답확인 개체 무결성(Entity Intergrity)

Text 정답확인 SNMP(Simple Network Management Protocol)

웹상에서 구글, 마이크로소프트, 아마존 등에서 제공하는 환경의 네트워크, 보안, 데이터 저장소, 콘텐츠 딜리버리 서비스를 포함한 다양한 인프라를 임대하여 이용할 수 있는 클라우드 서비스를 무엇이라고 하는가?

Text 정답확인 PaaS(Platform as a Service)

인프라와 운영체제뿐만 아니라 사용할 수 있는 소프트웨어까지 갖추어져 웹상의 로그인만으로 다양한 소프트웨어를 사용한 만큼 비용을 지불해 가며 사용할 수 있는 클라우드 서비스를 무엇이라고 하는가?
Text 정답확인 SaaS(Software as a Service)

부적절한 자료가 입력되어 동일한 내용에 대하여 서로 다른 데이터가 저장되는 것을 허용하지 않는 데이터베이스 관리스시템의 특징은 무엇인가?
Text 정답확인 데이터 무결성

Text 정답확인 무결성, 일관성, 회복성, 보안성, 효율성

미국 오라클사에서 개발한 데이터베이스 관리 시스템으로 리눅스/유닉스/윈도 모두를 지원하며 대형 시스템에서 많이 사용하는 관계형 데이터베이스 관리시스템은 무엇인가?
Text 정답확인 Oracle

정보를 일정한 크기로 분할한 뒤 각각의 패킷에 송수신 주소 및 부가정보를 입력한 것으로 현재컴퓨터 네트워크에서 주로 사용하는 방식은 무엇인가?
Text 정답확인 패킷 교환 방식

(         )는 하드웨어 기반으로 처리하기 때문에 속도가 빠르며, (         )는 소프트웨어 방식으로 처리하기 때문에 속도가 느리다. (         )는 포트들이 같은 속도를 지원하는 반면, ()는 각기 다른 속도를 지원하도록 제어할 수 있다. (         )는 제공하는 포트 수가 수십, 수백 개로 2~3개의 포트를 제공하는 브리지보다 많다. (         )는 Store and Forwarding 전송 방식만을 사용하나, (         )는 Cut Through와 Fragment Free 방식을 같이 사용한다.
Text 정답확인 스위치, 브리지

(         )는 여러 대의 컴퓨터를 연결하여 네트워크로 보내거나 하나의 네트워크로 수신된 정보를 여러 대의 컴퓨터로 송신하기 위한 장비이다. (         )는 디지털 신호를 증폭시켜 주는 역할을 하여 신호가 약해지지 않고 컴퓨터로 수신되도록 한다.
Text 정답확인 허브, 리피터

전기 통신 국제기구인 ITU-T에서 관리 감독하는 프로토콜이며, 데이터 블록을 사용하여 대용량의 데이터를 다수의 패킷으로 분리하여 송신하며, 수신 측에서는 다수의 패킷을 결합하여 원래의 데이터로 복원하는 프로토콜을 무엇이라고 하는가?
Text 정답확인 X.25

동기화를 맞추지 않아 보낼 데이터가 없는 사용자의 슬롯은 다른 사람이 사용할 수 있도록 하여 네트워크상의 효율성을 높이고 광대역 전송에 쓰이는 스위칭 기법은 무엇인가?
Text 정답확인 ATM(Asynchronous Transfer Mode)

패킷 스위칭과 달리 네트워크 리소스를 특정 사용층이 독점하도록 하여 네트워크를 독점적으로 사용하기 때문에 전송이 보장된다는 특징이 있다. 이러한 작업이 일어나는 동안 다른 기기들은 해당 경로를 사용할 수 없다. 이것을 무엇이라고 하는가?
Text 정답확인 서킷 스위칭
